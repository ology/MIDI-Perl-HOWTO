# MIDI-Perl HOWTO

How to create MIDI music with Perl!

    git clone https://github.com/ology/MIDI-Perl-HOWTO.git
    cd MIDI-Perl-HOWTO
    cpanm --installdeps .

Please visit the [wiki](https://github.com/ology/MIDI-Perl-HOWTO/wiki) for the tutorial.
